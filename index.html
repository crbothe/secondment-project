Dialogue-based Navigation with Multivariate Adaptation driven by Politeness

Click here to see the log of dialogues in the demo video

 in_utt = input utterance;
in_slu = input utterance spoken language understanding (dialogua act and slot-value pairs);
in_dop = input degree of politeness;
op_utt = output utterance (spoken by a robot);
INPUT_UTT = processed input utterance number (it is the utterance number in the graph);
NOT_INPUT = not processed input utterance (it is been used as a context only)
